================================================================================
FILE: taxonomy-interpretations.txt
Scope: jx-interpretation
Source: register-taxonomies.php v3.10.0
Notes: Full statute-mirror palette for interpretation records.
       Interpretations inherit the complete doctrinal taxonomy set from
       jx-statute. An interpretation is an applied reading of a statute —
       agency guidance, regulatory opinion, administrative ruling, or
       editorial doctrine note. Tag only what the interpretation genuinely
       addresses or clarifies; do not mirror parent statute tags by default.
       Many axes will often be unused or sparsely tagged on any single
       interpretation record; this file defines the allowed palette,
       not a required checklist.
================================================================================
