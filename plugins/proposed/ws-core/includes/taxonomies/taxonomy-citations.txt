================================================================================
FILE: taxonomy-citations.txt
Scope: jx-citation
Source: register-taxonomies.php v3.10.0
Notes: Full statute-mirror palette for citation records. Citations inherit
       the complete doctrinal taxonomy set from jx-statute. Many axes will
       often be unused or sparsely tagged on any single citation record;
       this file defines the allowed palette, not a required checklist.
       Tag only what is genuinely illustrated or narrowed by the cited case,
       ruling, or source — do not mirror parent statute tags by default.
================================================================================
