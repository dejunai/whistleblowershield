## `admin-major-edit-hook.php` — Findings

### 🟡 Should Fix

**1. `$law_name` operator precedence bug — `jx-citation` and `jx-interpretation` official names are unreachable**
The law name resolution block reads:

```php
$law_name = ( $post_type !== 'jx-summary' )
    ? get_post_meta( $post_id, 'ws_jx_statute_official_name', true )
    ?: get_post_meta( $post_id, 'ws_jx_citation_official_name', true )
    ?: get_post_meta( $post_id, 'ws_jx_interp_official_name', true )
    : '';
```

PHP parses the ternary `? :` before the `?:` null-coalescing chains . The actual evaluation is:

```php
$law_name = ( $post_type !== 'jx-summary' )
    ? get_post_meta( ..., 'ws_jx_statute_official_name', true )
    : ( get_post_meta( ..., 'ws_jx_citation_official_name', true )
        ?: get_post_meta( ..., 'ws_jx_interp_official_name', true )
        ?: '' );
```

So for a `jx-citation` or `jx-interpretation` post, only `ws_jx_statute_official_name` is tried — and it will always be empty on those CPTs, falling through to the `get_the_title()` fallback. The citation and interpretation official name fields are effectively dead code. Fix with explicit parentheses or a dedicated per-CPT lookup:

```php
$name_key = match( $post_type ) {
    'jx-statute'        => 'ws_jx_statute_official_name',
    'jx-citation'       => 'ws_jx_citation_official_name',
    'jx-interpretation' => 'ws_jx_interp_official_name',
    default             => '',
};
$law_name = $name_key ? get_post_meta( $post_id, $name_key, true ) : '';
if ( ! $law_name ) { $law_name = get_the_title( $post_id ); }
```

**2. `wp_insert_post()` fires `wp_insert_post` action, same hook-overlap risk as `ws_feed_ingest_item()`**
The auto-assign hook in `admin-hooks.php` will check `$_GET['ws_jx_term']` during the `ws-legal-update` post creation . `ws-legal-update` is not in the `$addendum_types` array so the hook bails immediately — no actual bug today. But if `ws-legal-update` is ever added to that array, the hook would fire and try to assign a taxonomy term from a stale GET param. Low risk, noted for awareness.
